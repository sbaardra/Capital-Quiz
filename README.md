# 🌍 Capital Quiz

A web-based quiz application to test knowledge of world capitals.

## Features

- User Login
- Region-wise Quiz
- Easy, Medium & Hard Levels
- Timer (15s, 10s, 7s)
- Lives System
- Combo Bonus Scoring
- Progress Bar
- Local Leaderboard
- Dark / Light Mode
- Sound Effects
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage
